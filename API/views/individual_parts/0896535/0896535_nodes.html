Here, you can manage the sensor nodes of the CHIBB house.

Create a sensor node:

Here you can create a new sensor node, and receive a token for your new sensor node.
If you create a new sensor node, a database table is automatically created for the sensor node, and you can start sending data to it with the token you receive here.

Turn a sensor node on or off:

Sets a sensor node to active or inactive, depending on it's current status.

Node overview table:

This table gives a overview of all of the sensor nodes, their and their status.
